# Chris Ogbonnaya — Portfolio

Premium personal portfolio built with React + Vite + Tailwind CSS + Framer Motion.

## Development

```bash
npm install
npm run dev
```

Runs on http://localhost:8080

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

Push to GitHub and import the repo in Vercel. Framework preset: **Vite**. No extra configuration needed — `vercel.json` handles SPA fallback.

## Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4
- React Router DOM
- Framer Motion
- Lucide Icons
- shadcn/ui
